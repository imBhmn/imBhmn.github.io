  خب اگر این رو خالی بزارین یه فاز ولنگ و باز میگیره 
  بهتر بگم،ما تایپ های مختلفی رو داریم 
  - رشته / string: دوست عزیزی که فقط رشته میگیره
  - عدد / number: دوست عزیز تری که فقط عدد میگیره
  - نمیدونم /boolean: دوست عزیزی که true یا false میگیره
  - همه چی /any: این دوست عزیزمون همه چی میگیره
  - جای گذار؟ / generic : این همون کار any رو میکنه با این تفاوت که وقتی برای ورودی و خروجی فانکشن میاد بهمون کامند های بقیه تایپ ها مثل string یا number رو میده
